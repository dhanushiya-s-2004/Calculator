# 🧮 Simple Calculator

## 📖 Project Overview

This is a simple command-line calculator developed using Python. It performs four basic arithmetic operations based on the user's choice.

---

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🚫 Prevents division by zero
- ✅ Handles invalid menu choices

---

## 🚀 How to Run

### Clone the repository

```bash
git clone https://github.com/dhanushiya-s-2004/Calculator.git
```

### Move to the project folder

```bash
cd Calculator
```

### Run the program

```bash
python Calculator.py
```
